API Overview

The LimeWP API allows you to programmatically manage your WordPress sites, backups, domains, and more. Our RESTful API uses standard HTTP methods and returns JSON responses.

Base URL https://api.limewp.com/v1
Authentication

All API requests require authentication using an API key. Include your API key in the request header as shown below. Keep your API key secure and never expose it in client-side code.

Quick Start Guide

5 min read
Updated Jan 15, 2026

Welcome to LimeWP! This guide will help you get started with your WordPress hosting in just a few minutes. Follow these steps to launch your first site.

Before you begin

Make sure you have your domain name ready. If you don't have one, you can purchase one through our dashboard or use a temporary LimeWP subdomain.

Step 1: Create a New Site

From your dashboard, click the Add New Site button. You'll be prompted to enter your site details including:

  • Site Name: A friendly name for your site (e.g., "My Blog")
  • Domain: Your custom domain or a free LimeWP subdomain
  • WordPress Version: We recommend using the latest stable version
  • PHP Version: PHP 8.1 or higher is recommended

Step 2: Configure Your Site

After your site is created, you can access the WordPress admin panel to customize your installation:

# Access your WordPress admin at:
https://yourdomain.com/wp-admin

# Default credentials are sent to your email
# We recommend changing your password immediately
Security Reminder

Always use strong passwords and enable two-factor authentication for your WordPress admin account.

Step 3: Install Essential Plugins

We recommend installing these plugins to enhance your site's functionality and security:

Security Plugin
Protect your site from malware and unauthorized access
Caching Plugin
Speed up your site with advanced caching features
Image Optimizer
Automatically compress images for better performance
SEO Plugin
Optimize your site for search engines
You're all set!

Congratulations! Your WordPress site is now ready. Explore our documentation to learn more advanced features.
